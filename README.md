## 🚀 Project Overview

This project consists of two main components:

1. **Investment Calculator**

   * Calculates returns using compound interest
   * Supports Lump Sum and SIP investments
   * Displays maturity value, gains, CAGR, and inflation-adjusted value

 **Investment Dashboard**

   * Shows portfolio overview
   * Displays KPIs like total value, returns, and daily profit
   * Includes charts, sector allocation, holdings, and recent orders

---

## ✨ Features

### 📊 Calculator Features

* Lump Sum & SIP calculation
* Real-time updates with sliders
* CAGR & inflation-adjusted returns
* Year-by-year growth table
* Visual bar chart

### 📈 Dashboard Features

* Portfolio summary (value, returns, P&L)
* Interactive growth chart
* Sector allocation (donut chart)
* Top holdings display
* Recent transactions/orders
* Clean and modern UI

---

## 🛠️ Technologies Used

* **HTML5** – Structure

## ⚙️ How to Run

1. Download the project files
2. Open in browser:

   * `investment-calculator.html`
   * `investment-dashboard.html`
3. No installation required 🚀
